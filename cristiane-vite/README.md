# Dra. Cristiane Rodrigues — Site

Vite + React + TypeScript SPA.

## Desenvolvimento
```
npm install
npm run dev
```

## Build
```
npm run build
```
Gera `/dist` pronto para deploy.

## Deploy no Vercel
1. Push para o GitHub
2. Importe o repositório no Vercel
3. Framework: Vite (auto-detectado). `vercel.json` já configura o rewrite SPA.
